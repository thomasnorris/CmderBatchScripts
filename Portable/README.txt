These files will download Cmder and config files from Dropbox and Cmder's GitHub page.
7-zip portable is required to be in this directory in a folder called 7-zip.
7-zip can be downloaded here: http://www.7-zip.org/download.html (download the "Extra" one).

If there is a path issue starting Cmder for the first time...
Copy this next line and paste it at the top of cmder_install_location\vendor\init.bat after @echo off
set PATH=%PATH:"=%

Restart Cmder.
